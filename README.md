# atv-Lista1-Python

## Descrição
Este repositório contém a atividade de Python, passada para praticar lógica de programação.

## Questões
1 – Escreva um programa que receba 2 valores do tipo inteiro x e y, e calcule o valor de z = (𝑥²+ 𝑦²) x (𝑥−𝑦)².
2 - Escreva um programa que receba o salário de um funcionário (float), e retorne o resultado do novo salário com reajuste de 35%.
3 – Faça um programa que leia 2 notas de um aluno, calcule a média e imprima aprovado ou reprovado (para ser aprovado a média deve ser no mínimo 6).
4 – Refaça o exercício 3, identificando o conceito aprovado (média superior a 6), recuperação (média entre 4 e 6) ou reprovado (média inferior a 4).
